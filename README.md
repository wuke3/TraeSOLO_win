# TRAE SOLO 桌面包装器

TRAE SOLO 桌面包装器是一个基于 Electron 开发的跨平台桌面应用程序，用于包装 TRAE SOLO 的网页版，提供更原生的桌面使用体验。

## 功能特性

### 核心功能
- **跨平台支持**：支持 Windows 和 Linux 操作系统
- **自定义窗口**：无边框窗口设计，带有自定义标题栏和控制按钮
- **服务器切换**：可在国内版 (solo.trae.cn) 和国际版 (solo.trae.ai) 之间切换
- **主题切换**：支持深色模式和浅色模式，并能自动检测网页主题
- **预约功能**：内置预约 Windows 版本的弹窗
- **外部链接处理**：外部链接在新的自定义窗口中打开
- **文件下载**：直接选择文件保存位置，无需打开浏览器

### UI 特性
- **统一按钮样式**：所有按钮采用一致的设计风格
- **禁用标题栏右侧按钮**：顶部标题栏右侧的按钮已禁用，保持界面简洁
- **响应式设计**：适应不同屏幕尺寸

## 技术栈

- **Electron**：跨平台桌面应用开发框架
- **HTML5**：页面结构
- **CSS3**：样式设计
- **JavaScript**：交互逻辑

## 安装步骤

### 前提条件
- Node.js (v14.0.0 或更高版本)
- npm (v6.0.0 或更高版本)

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm start
```

### 构建应用程序

#### 构建 Windows 版本

```bash
npm run build
```

构建产物将生成在 `dist` 目录中。

## 使用说明

### 服务器切换
1. 点击顶部标题栏的设置按钮
2. 在设置面板中选择 "国内版 (solo.trae.cn)" 或 "国际版 (solo.trae.ai)"
3. 应用程序会自动切换到选择的服务器

### 主题切换
1. 点击顶部标题栏的设置按钮
2. 在设置面板中使用深色模式开关
3. 应用程序会自动切换到选择的主题
4. 应用程序也会自动检测网页的主题模式并同步调整

### 预约 Windows 版本
1. 点击顶部标题栏的 "预约" 按钮
2. 在弹出的预约窗口中填写相关信息
3. 点击 "提交" 完成预约

### 处理外部链接
- 当点击网页中的外部链接时，会自动在新的自定义窗口中打开
- 外部链接窗口也具有与主窗口相同的控制功能

### 下载文件
- 当在网页中点击下载链接时，会弹出文件保存对话框
- 选择保存位置后，文件会直接下载到指定位置

## 常见问题

### 应用程序无法启动
- 检查 Node.js 和 npm 是否正确安装
- 确保所有依赖已成功安装
- 检查网络连接是否正常

### 无法切换服务器
- 检查网络连接是否正常
- 确保选择的服务器地址可访问

### 主题切换不生效
- 尝试刷新网页
- 检查网页是否支持主题切换

## 故障排除

### 查看应用程序日志
- 在开发模式下，控制台会显示详细的日志信息
- 可以通过开发者工具查看更多调试信息

### 重置设置
- 应用程序的设置存储在本地，可以通过删除配置文件来重置设置
